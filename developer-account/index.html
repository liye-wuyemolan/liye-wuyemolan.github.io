    注册开发者账号

    开发者账号注册认证

    开放平台开放诸多能力和服务， 助力开发者打造优质应用。开发者需要注册蓝卓账号，并且实名认证才能享受平台开放的各类能力和服务。

    1. 注册、认证准备的资料¶

    注册、认证蓝卓帐号需要准备的资料如下：

    账号类型 服务类型 所需材料
    个人/企业 注册 可接收验证码的手机号码
    个人 人工审核 身份证原件正反面扫描件图片
    企业 人工审核 1.法人代表身份证原件正反面扫描件图片；
    2. 企业四证合一扫描件图片

    2. 实名认证审核时间¶

    开放凭条将在1至2个工作日内完成审核。审核完成后，将向您提交的认证信息中的联系人手机号码发送审核结果通知。

    3. 账号注册与登录¶

    打开 开放平台 首页，页面右上角有“立即注册”以及“登录”按钮。

    3.1. 注册账号

    如果您已经有蓝卓帐号，点击右上角的登录按钮，登录开放平台。

    如果你还没有注册蓝卓账号，请先点击右上角导航栏中的“立即注册”按钮，进行账号注册。

    账号注册使用“手机号”方式注册

    • 在注册页面，输入您的有效手机号（中国大陆手机号）、进行滑块验证、短信验证码、密码以及密码确认（两次密码需一致）；
    • 最后确认信息无误后点击“注册”按钮，提交注册信息。
    • 注册成功后将显示账号生成提示页面，显示生成的Bluetron ID，并提供跳转实名认证页面以及开放平台首页链接按钮。

    手机号已被注册使用？

    注册Bluetron ID 时，输入手机号码，若提示，该手机号码已被使用，则说明当前输入的手机号已经注册绑定过Bluetron ID，个人开发者请自行确定是否自己或曾授以他人在开放平台注册过账号，公司开发者请先内部确定是否其他同事再开放平台注册过。每个手机号码只能绑定一个开发者账号

    手机号一经注册不可变更

    建议使用公司指定手机号而不是员工私人号码注册，以免遇到员工离职等情况造成不必要的麻烦。

    3.2. 登录账号

    蓝卓账号施行一体化机制，应用商店帐号、开放平台帐号均可登录开放平台。

    如您已有蓝卓账号（应用商店注册账号亦可直接登录）可以直接点击右上角“登录”按钮，登录开放平台。

    • Bluetron ID（蓝卓账号），默认为之前注册页面填写的手机号码，注册成功后的页面提示信息中也有显示，请参照上述注册相关说明
    • 输入已有的账号密码，点击“登录”即可。
    • 如果您未注册过蓝卓账号，当前页面提供“免费注册”快速链接到账号注册页面。

    4. 实名认证¶

    4.1. 认证入口

    • 注册成功后，在页面上点击“立即实名认证”，进入开发者认证界面；
    • 开放平台首页，点击“申请成为开发者”，进入认证界面；
    • 登录成功后，鼠标移动至“欢迎XXX”上，下滑展示账号信息区域，点击“实名认证”链接，进入开发者认证界面；

    4.2. 认证状态

    • 未认证
      当前账号注册成功并登录后，进入认证页面，将提示您选择认证类型：企业开发者认证、个人开发者认证。

    • 认证审核中
      正确填写实名认证资料后，进入认证审核阶段，再次点击认证入口进入页面，跳转提示信息页面，会显示当前认证流程步骤环节，以及当前账号认证类型等信息。

    • 实名认证审核通过
      工作人员根据提交的认证资料审核通过后，将通过短信的形式告知到您，根据提示进入开放平台首页，点击认证入口快速链接，跳转如下页面。

    • 实名认证审核失败
      工作人员根据提交的认证资料审核后，将通过短信的形式告知到您，根据提示进入开放平台首页，点击认证入口快速链接，跳转如下页面进行查看失败原因，并点击“重新申请认证”更新提交资料进行审核。

    4.3. 鉴权

    • 未登录用户可以访问开放平台的静态首页面，当试图进入“开发者中心”页面时，将强制跳转到登录页面；
    • 未提交实名认证（或认证被驳回）的开发者账号，成功登录系统后，当试图进入“开发者中心”页面时，将强制跳转到实名认证页面；
    • 已提交实名认证资料，等待审核的开发者账号，成功登录系统后，当试图进入“开发者中心”页面时，将强制跳转到提醒等待审核页面；

    4.4. 认证资料填写

    4.4.1 企业实名认证

    代表企业开发者，点击“我是企业开发者”，进入资料填写页面，展示如下：

    字段名称 填写说明 备注
    企业全称 当前注册账号所属公司名称 必填，与企业认证图片中名称保持一致
    企业地址 当前注册账号所属公司地址 必填，与企业认证图片中名称保持一致
    联系人姓名 联系人姓名 必填
    手机号码 联系电话 必填，主要用于接收通知类短信
    短信验证码 点击“获取验证码”获取，根据短信提示输入信息 必填，验证联系人手机号码有效性
    联系人邮箱 联系人邮箱地址或者企业邮箱地址 必填，用于接收通知公告等信息
    法人代表身份证扫描件 企业法人代表身份证正面（头像面）以及背面（国徽面）图片文件 必填，据提示对应正面与反面各自上传，并保证上传的图片清晰可见，水平拍摄的图片类型文件
    企业四证合一扫描件 企业合法证件扫描图片文件上传 必填，上传真实有效的企业注册信息扫描件，并与上述填写的企业全称与企业地址信息保持一致
    上一步 按钮操作 返回选择认证类型页面
    下一步 按钮操作 页面信息提交进入审核阶段

    4.4.2 个人实名认证

    代表个人开发者，点击“我是个人开发者”，进入资料填写页面，展示如下：

    字段名称 填写说明 备注
    姓名 联系人姓名 必填，开发者真实姓名，与证件名称保持一致
    手机号 联系人手机号 必填，默认绑定当前账号注册手机号，可变更，主要用于后期接收通知类短信信息
    短信验证码 点击“获取短信验证码”获取，根据短信提示操作 必填，验证联系人手机号的真实有效性
    固定电话 联系电话 非必填
    邮箱 联系人邮箱地址 非必填
    所属公司 联系人所属公司名称 非必填
    证件类型 默认选择身份证
    身份证号 联系人身份证号码 必填，证件号码与上传的身份证图片正面扫描件中的号码保持一致
    身份证扫描件 联系人身份证正面（头像面）以及背面（国徽面）图片文件 必填，与证件号码保持一致，并根据提示对应正面与反面各自上传，并保证上传的图片清晰可见，水平拍摄的图片类型文件
    上一步 按钮操作 返回选择认证类型页面
    下一步 按钮操作 页面信息提交进入审核阶段

    4.4. 等待审核

    提交资料后，我方工作人员将会在1-3个工作日对您提交的资料进行审核验证，期间可能会有相应的短信验证或者邮箱验证，敬请留意信息内容。审核结果将第一时间通过短信方式告知当前注册账号对应的手机上，请留意相关短信内容。

    4.5. 后续操作

    开发者可以在“开发者控制台” > “账号信息” 查看已验证的身份信息。
